This uses the Leaflet JavaScript library with a static map image, the NPS park brochure for Gettysburg National Park. Pins on the map match the stops on the vehicle tour and have links to wikipedia articles. 

The park brochure was downloaded from the Library of Congress site: http://hdl.loc.gov/loc.gmd/g3822g.cw0352500

The basic_map.html map was added to show the larger geographic context of Gettysburg. It was created using the Quickstart tutorial on the leafletjs site https://leafletjs.com/examples/quick-start/ Links were added to each map to allow toggling. The leaflet map shows geo-coordinates when clicking on the map and would allow making Google maps links.
